Sail 環境中，針對 MySQL Container 給 sail 用戶權限

Created At：2022-06-27 10:24 Updated At：2022-06-27 10:30
Keywords: sail

日期：2022-06-27 ( 一 )

正常來講 Sail 環境中裡的 MySQL 容器，裡面有一個 sail 用戶，只能操作自己的資料庫，無法新增其它資料庫，或更名資料庫等操作

查看專案目前使用的 container

進入 mysql 那個 container

docker-compose exec mysql bash

參考 .env 設定檔

猜測 root 和 sail 用戶的密碼都一樣為 .env 所設定的 password

mysql -u root -p

給 sail 用戶所有權限

GRANT ALL PRIVILEGES ON *.* TO 'sail'@'%';

刷新權限

FLUSH PRIVILEGES;

然後登出

再使用 PHPMyAdmin 就可以進行資料庫新增、更名的動作了
