PHPStorm 常用設定 macOS 版本

Created At：2020-03-16 15:39 Updated At：2020-05-11 09:35

有一些不錯的設定可以用來增加開發效率，或減少出錯的機率

Windows 的人要自己找一下…

顯示、隱藏 function 與 function 之間的分隔線

搜尋 separators 分隔符，找到 Show method separators

手動一次選取多個單字的方法

Mac 版本：按住 ⌥ Option 鍵，然後用滑鼠點選 2 次要選擇的單字

分欄顯示程式碼

Mac 版本：按住 Control（或 Ctrl）鍵 ⌃ 然後點選檔名叫出選單，選擇 Split Vertically ( 垂直分欄 ) 或者 Split Horizontally ( 水平分欄 )

自訂程式碼快速鍵

自訂快速鍵，叫出自訂片段程式碼，功能區塊為 Live Templates

安裝新樣式、新字體

教學 https://www.youtube.com/watch?v=3SUtEnMj1ws

material 佈景主題 https://plugins.jetbrains.com/plugin/8006-material-theme-ui

Jetbrains 的字體 https://www.jetbrains.com/lp/mono/

空隔顯示設定

Class 的註解

Preferences > Editor > File and Code Templates > Includes tab > PHP Class Doc Comment

註解效果：Class

Class 裡 Function 的註解

Preferences > Editor > File and Code Templates > Includes tab > PHP Function Doc Comment

註解效果：Class 裡的 Function
